# 🛡️ Strategic Intervention Engine (S.I.E.)
### *AI-Powered Propensity Modeling for Frictionless Asset Recovery*

## 📌 Executive Summary
The **Strategic Intervention Engine** is an operations-first prototype designed to move organizations from reactive collections to proactive, data-driven "Member Support." By leveraging **Propensity-to-Pay (PTP)** logic and **Best-Time-to-Contact (BTC)** heuristics, the engine identifies a borrower's "Golden Window" for engagement—reducing friction and protecting institutional yield.

## 🚀 Key Features
* **Propensity-Driven BTC Intelligence:** Intuitively detects optimal outreach triggers based on historical engagement patterns, industry behavior, and risk intensity.
* **Predictive "Hiccup" Scoring:** Multi-feature decision trees identify high-risk accounts before they reach late-stage delinquency.
* **Empathetic AI Payload:** Generates partner-focused communication designed to preserve member relationships while maximizing recovery.

## 🛠️ Technical Stack
* **Framework:** Streamlit (Python)
* **LLM Integration:** LangChain & Ollama (Llama3)
* **Data Processing:** Pandas
* **Logic:** Heuristic-based Propensity Modeling

## 📂 Deployment & Usage
To run locally with AI enabled:
1. Ensure [Ollama](https://ollama.com) is installed and running `llama3`.
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `streamlit run collections_intervention_app.py`
